# 使用技術
Ruby 2.5.7, Rails 5.2.3

# 機能一覧
ユーザー登録・ログイン機能 (ユーザーのプロフィール画像のアップロードにActive Storageを使用)
投稿機能（画像のアップロードにCarrierWave, を使用）
登録された住所をマップに表示する機能(Google Apiを使用)
投稿一覧・投稿詳細表示機能
投稿管理機能
ページネーション機能（Kaminari）
いいね機能（Ajax）
レビュー機能（Ajax）
検索機能（Ransackを使用）
